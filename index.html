Моё портфолио

Главные личные проекты:

- [netstalker] (в разработке) -- данное приложения является Pet проектом, который призван отразить максимальное кол-во мои знаний в среде Web разработки. Приложение пишется на React JS с использованием Bootstrap для стилизации. В будущем планируется реализовать регистрацию и авторизацию на сервисе, полноценные личные кабинеты пользователей, возможность создавать и размещать статьи для общего доступа и другое;

- Калькулятор -- простой калькулятор, логика которого была написана с мной нуля;

- Change Color -- это приложение было написано по курсу на YouTube и является первым полноценным приложением. В этом приложении можно настроить собственную цветовую гамму и затем поделиться ей с другим человеком, посредством копирования ссылки в адресной строке. В этой ссылке встроены выбранные цвета и при переходе по ссылке открывается выбранная отправителем цветовая гамма.

Художественные проекты (пример адаптивной вёрстки с разнообразными визуальными эффектами):
